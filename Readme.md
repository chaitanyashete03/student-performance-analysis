# 📊 Student Performance Data Analysis

## 🔍 Project Overview
This project analyzes student performance data to uncover patterns between:
- Study hours
- Gender
- Academic scores

The goal is to demonstrate practical data analysis skills using Python.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📈 Key Analysis Performed
- Distribution of student marks
- Gender-wise average marks
- Relationship between study hours and performance

---

## 💡 Insights
- Study hours show a positive correlation with marks
- Gender-wise averages highlight performance differences
- High scorers are not always those with extreme study hours

---

## 📁 Files in This Repository
- `analysis.ipynb` → Complete analysis notebook
- `student_data.csv` → Dataset
- `README.md` → Project documentation

---

## 🚀 Future Improvements
- Add statistical correlation analysis
- Include more features like attendance
- Convert insights into a dashboard

---
